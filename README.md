# String 타입 덧셈 

## 준비사항 
* JDK Version >=8
* IntelliJ or Eclipse

## 학습내용 
* int 타입, Long 타입으로도 표현하지 숫자는 매우 많이 존재한다. 
그러한 숫자들 끼리의 사칙연산을 위해 BigInteger, BigDecimal 등이 존재한다. 
BigInteger 를 사용하지 않고 문자열 타입끼리의 덧셈을 하는 방법을 구현하기 위해 코드를 작성

- [x] 글자수를 맞추어서 덧셈 
- [x] add 함수 쓰지않고 main에서 구현 
- [x] 글자수 맞추지 않고 자동으로 맞추게 설계
